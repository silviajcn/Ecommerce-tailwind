## ⚙️ Installation

* Clone the latest code on the master branch
    * ```git clone https://github.com/silviajcn/NotesApp.git```
* Install dependencies
    * ```npm install```
* Run the web server!
    * ```npm run dev```

## 🎈 Usage

* Navigate to http://localhost:5173 on your browser
